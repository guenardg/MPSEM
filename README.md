# R Package MPSEM

Package MPSEM allows on to generate Phylogenetic Eigenvector Maps (**PEM**), which
are used to represent phylogenetic signals in data sets involving multiple species.

When observations are made on multiple species, observations made on a given
species are more dependent from one another than observations made on different
species. Moreover, observations performed on two related species are more
dependent from one another than observations performed on distant species, since
species share different levels of parenthood. This heterogeneity in relatedness
can be quantified using information on the phylogeny (think: the tree of life) and
a maximum likelihood model and put to use in describing patterns of phylogenetic
variations (spectral method) and predicting unknown species traits from those of
related species with known trait values.

Applications in ecotoxicology, ecophysiology, and paleothology have been documented.
